Cài đặt Zsh và Terminator

Zsh và Terminator mang lại cho mình rất nhiều tiện tích trong các công việc cần đến Terminal. Vì vậy, sau khi cài đặt Ubuntu hay LinuxMint thì bạn nên cài Zsh và Terminator đầu tiên.

Cài đặt Zsh
sudo apt-get install zsh  
# Make sure it installed by running 
zsh --version  
# Make it your default shell
chsh -s $(which zsh)
Cài đặt Oh-My-Zsh
sh -c "$(wget https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh -O -)"
Cuối cùng là cài Terminator
sudo add-apt-repository ppa:gnome-terminator  
sudo apt-get update  
sudo apt-get install terminator  
# Start terminator
terminator

Bây giờ chúng ta có thể khởi động Terminator và dùng thử, với terminator chúng ta có thể split 1 cửa sổ ra thành nhiều terminal khác nhau để tiện dụng cho việc sử dụng. Chỉnh được các phím tắt, như mình thường chỉnh Ctrl + C cho Copy, Ctrl + V cho Paste cho giống với iTerm2 trên MacOS.

Để tiện dụng hơn nữa các bạn có thể chỉnh phím tắt trong hệ thống để khi nhấn Ctrl + Alt + T có thể bật 1 cửa sổ Terminator mới cho tiện.

Nguồn bài viết - http://kipalog.com/posts/Cai-xong-Linux-Mint-Ubuntu-roi-thi-lam-gi-tiep

*Bonus tip: Một trong những kỹ thuật hay dùng nhất với Dev khi thao tác trên Terminal là đặt alias cho các command mình thường xuyên sử dụng. Trong Zsh, Alias được đặt trong file .zshrc (tương tư .bashrc)
